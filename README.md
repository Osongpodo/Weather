## ⚙ 설정 방법
**🌧 Rain을 좀 더 Heavy하게**

- Emission - Rate over Time을 증가시킬수록 Heavy

**☃ Snow를 좀 더 Heavy하게**

- Emission - Rate over Time을 증가시킬수록 Heavy

**☁ Fog를 좀 더 짙게**

- Color over Lifetime - Color - 가운데 태그의 Alpha 값을 조정
